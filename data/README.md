# 数据说明

本目录存放 MapFinBen 项目的数据文件，当前按照五个任务分别组织为以下子目录：

- `CroFinAS`：金融选择问答数据，主要用于评估模型在金融场景下的选择题理解、选项判别与推理能力
- `CroFinQA`：金融文本问答数据，主要用于评估模型基于金融文本进行信息抽取、阅读理解与问答生成的能力
- `CroFinSA`：金融情感分析数据，主要用于评估模型对金融文本中积极、中性、消极等情感极性的识别能力
- `CroFinTC`：金融主题分类数据，主要用于评估模型对金融文本主题类别的识别与分类能力
- `CroFinTS`：金融文本摘要数据，主要用于评估模型对金融文本核心信息的压缩、归纳与摘要生成能力

各任务目录下目前包含以下三类 JSON 文件：

- `*_train.json`：训练数据
- `*_valid.json`：验证数据
- `*_test.json`：测试数据

其中，测试集文件当前不纳入 Git 版本管理；评测结束后，将统一公开对应测试集数据。

当前 Git 忽略规则仅针对 `data` 目录下的 `*_test.json` 文件生效，其余数据文件可按需要加入版本管理。

如需查看各任务的数据字段、样例和更完整的任务说明，请结合项目根目录下的 [`README.md`](../README.md) 一并阅读。
